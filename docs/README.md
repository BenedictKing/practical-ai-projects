# 项目文档

本目录包含 My Best Practice 项目的完整规划与设计文档。

## 核心文档

- **[PROJECT.md](PROJECT.md)** - 项目定义、核心价值、约束条件与关键决策
- **[REQUIREMENTS.md](REQUIREMENTS.md)** - v1/v2 需求规格与追溯矩阵
- **[ROADMAP.md](ROADMAP.md)** - 三阶段开发路线图

## 阶段计划

[phases/](phases/) 目录包含各阶段的详细计划与执行记录：

- **Phase 1: 定位与主页骨架** - 建立清晰可信的单页入口
- **Phase 2: 项目展示与继续浏览** - 让访客快速理解项目并进入详情
- **Phase 3: README 同源精简版** - GitHub 访客也能快速理解定位

## 运行时状态

项目执行状态由 `.planning/STATE.md` 维护（不纳入版本控制）。

## 文档演进

- 项目定义文档在阶段转换和里程碑边界时更新
- 需求文档在路线图创建后锁定追溯关系
- 路线图在阶段完成后更新进度
